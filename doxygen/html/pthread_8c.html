Waits for the given thread the finish, and places the value given from that thread in pthread_exit into retval.

Parameters
threadThread created with pthread_create.
retvalPointer to the retval pointer that pthread_exit can fill upon exiting. NULL if no return value needed.
Returns
If the process does not have a pthread running with the given thread object or the process has been killed, will return -1. Otherwise, will return the pid of the exiting thread.
